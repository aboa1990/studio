# **App Name**: ForgeDocs Management

## Core Features:

- Document Management: Create, view, and edit invoices and quotations with automated numbering, MVR currency, and real-time tax calculation.
- PDF Export: Generate and download professional PDF versions of invoices and quotations directly from the application.
- User Authentication & Authorization: Secure login system with distinct user roles (e.g., Admin, Staff) managing access to an administrative panel.
- Firebase Data Persistence: Utilize Firebase Firestore for secure, real-time storage of all company data, documents, and user profiles.
- Dashboard Analytics: A dynamic dashboard providing an overview of key business metrics, financial summaries, and document status.
- AI-Powered Email Composer Tool: A generative AI tool to assist in drafting professional, personalized email content for sending invoices and quotations to clients.
- Company Branding Configuration: Admin tools to upload company logos and set business details that automatically appear on all generated documents.

## Style Guidelines:

- The application employs a sophisticated dark color scheme to convey professionalism and reduce eye strain for extended use. The primary color is a confident blue, `#8FC8F7`, used for interactive elements and highlights, providing excellent contrast against darker backgrounds. The background color is a deep, desaturated blue-gray, `#141A1F`, creating a focused and modern visual environment. An accent color of vibrant cyan, `#3DD5F5`, is used sparingly for calls to action and critical notifications, ensuring visibility and user attention.
- All text within the application, including headlines and body content, will utilize 'Inter', a modern sans-serif typeface. This choice provides exceptional legibility across various screen sizes and ensures a clean, professional, and consistent textual experience suitable for a business management system.
- Utilize a consistent set of crisp, vector-based icons that adhere to a minimalist and professional design. Icons should be clear and intuitive, supporting document actions (e.g., Create, Edit, Delete, Download) and navigation elements to enhance user experience without clutter.
- The layout is designed for clarity and efficiency, prioritizing logical content hierarchy and ease of access. It will feature a clean dashboard structure and responsive design to ensure optimal viewing and interaction across desktop, tablet, and mobile devices.
- Subtle and purposeful animations will be integrated to provide immediate user feedback on interactions, such as form submissions, data loading, or navigation transitions. These animations will be fast, non-intrusive, and contribute to a smooth, premium feel of the application.
